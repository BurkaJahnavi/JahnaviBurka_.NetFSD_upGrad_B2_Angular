1. What is jQuery?
jQuery is a JavaScript library that simplifies DOM manipulation and event handling.
2. Why use jQuery?
It makes coding easier, shorter, and cross-browser compatible.
3. What is slideToggle()?
slideToggle() smoothly shows or hides elements with sliding animation.
4. What is DOM manipulation?
It means modifying HTML elements dynamically using JavaScript or jQuery.
5. What is event handling?
It allows actions to be performed when users click or interact with elements.
